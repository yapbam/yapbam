Version 0.8.1 (05/03/2011)

Nouveautés de cette version

  • Correction d'un bug : Le changement de pays/langue ne changeait pas certains format de date.
  • Correction d'un bug : Lorsque d'anciennes opérations n'étaient pas pointées, le relevé "Non pointé" pouvait ne pas être le plus récent dans l'onglet "Relevés de compte". Ceci rendait les soldes en début et fin de périodes faux pour les relevés situés avant le relevé "Non pointé".
  • Le camembert de dépenses par catégorie est maintenant trié par libellé croissant de catégorie.
  • Ajout d'un bouton "Tutoriel" dans l'écran d'accueil.

Versions précédentes

Version 0.8.0 (04/02/2011)

  • Ajout de nouveaux thèmes visuels (Voir le dialogue accessible par le menu "Fichier/Préférences", onglet "Apparence").
    Nous vous recommandons de tester l'apparence "Nimbus".
  • Ajout d'un écran de bienvenue, regroupant des raccourcis utiles et des astuces d'utilisation de Yapbam.

  • Correction d'un bug : Ne plus autoriser un mode de paiment pour un type d'opération (recette/dépense) pour lequel il était précédemment utilisé pouvait provoquer des erreurs.
  • Correction d'un bug : Créer une catégorie dans une sous-opération pouvait changer la catégorie de l'opération.
